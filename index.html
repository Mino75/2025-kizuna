Kizuna

Website Enhancement Injection Service

Enhance Your Website with Kizuna

Kizuna is a powerful injection service that adds interactive features to any website.

Available Features

Auto Scroll

Intelligent scrolling functionality with zoom awareness

Interactive Quizzes

Markdown table-based quiz system with beginner and advanced modes

JavaScript Sandbox

Safe code execution environment for testing and validation

Privacy Compliance

GDPR-compliant privacy popups and information management

How to Use

Add Kizuna to your website by including this script tag:

<script defer src="https://kizuna.kahiether.com/main.js" website-url="https://yoursite.com" personalinfostored="no" enablemdquizz="yes" enablejssandbox="no" enableprivacy="yes"></script>

Configuration Parameters

website-url: Your website URL (for privacy notices)
personalinfostored: "yes" or "no" - Whether your site stores personal info
enablemdquizz: "yes" or "no" - Enable quiz functionality
enablejssandbox: "yes" or "no" - Enable JavaScript sandbox
enableprivacy: "yes" or "no" - Enable GDPR privacy popup
